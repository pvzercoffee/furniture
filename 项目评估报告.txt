【项目概述】

项目名称: Furniture 家具商城系统
项目类型: 前后端分离的Web应用
开发者: pvzercoffee
项目定位: 家具咨询与留言系统

本项目是一个基于现代Web技术栈构建的家具商城咨询系统，主要功能包括用户注册登录、
浏览家具产品信息、留言咨询以及个人信息管理。项目采用前后端分离架构，具有较好的
可扩展性和维护性。

【快速启动指南】
================================================================================

一、环境要求
-------------------------------------------
前端：
- Node.js: ^20.19.0 或 >=22.12.0
- npm: 最新版本

后端：
- JDK: 17
- Maven: 3.6+
- MySQL: 5.7+


二、启动步骤
-------------------------------------------

1. 数据库准备
   - 创建数据库: furniture
   - 执行 sql/user.sql 初始化用户表
   - 执行 sql/message.sql 初始化留言表

2. 后端启动
   cd furniture-server
   mvn clean install
   mvn spring-boot:run

3. 前端启动
   cd furniture-brownser
   npm install
   npm run dev

4. 访问应用
   浏览器打开: http://localhost:5173/furniture
